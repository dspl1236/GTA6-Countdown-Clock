## Case

A 3D-printable desk-wedge enclosure is planned (design in progress). STL/STEP
files will land here when ready. Community case designs welcome — see
`CONTRIBUTING.md`.

Design intent: solid wedge body with an ~18° tilted face, the OLED framed in a
bezel opening, the ESP32 housed in the base, cable exit at the lower rear.
Prototype in PLA/PETG, finish in translucent resin so the body glows with the
screen.
